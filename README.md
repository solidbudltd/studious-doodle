# SOLIDBUD LTD – Company Website

Professional company website for **SOLIDBUD LTD**, a construction firm offering residential, commercial, and industrial building services.

## Features

- **Responsive design** – works on desktop, tablet, and mobile devices
- **Hero section** – impactful landing area with company stats
- **About section** – company overview with key strengths
- **Services section** – 6 core service offerings with icons
- **Projects section** – filterable portfolio of completed projects
- **Testimonials** – client reviews
- **Contact form** – validated form with UX feedback
- **Smooth navigation** – sticky header, smooth scroll, active link tracking
- **Back-to-top button** – for easy navigation

## Tech Stack

- HTML5 (semantic markup, accessibility attributes)
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (ES2020, no frameworks or dependencies)
- Google Fonts (Inter)

## Running Locally

Simply open `index.html` in any modern web browser. No build step required.

```bash
open index.html
# or
python3 -m http.server 8080
```

## Language

The website content is in **Polish** — SOLIDBUD LTD is a Polish-run construction company based in Manchester, UK, serving the Polish-speaking community in Greater Manchester. All pricing is in GBP (£) and VAT is at the UK standard rate of 20%.
