# Dom & Ogród Inwestycyjny — Czysto i Świeżo

Single-page static website for a low-budget investment property with garden.

**Motto:** *Czysto i świeżo* (Clean & Fresh)

## Files

| File | Description |
|---|---|
| `index.html` | Single-page site: hero, feature cards, photo gallery, contact form |
| `style.css` | Minimal green/white palette stylesheet |

## Running locally

Open `index.html` directly in any browser — no build step required.
